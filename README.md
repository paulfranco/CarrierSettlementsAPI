
# CarriersApp API

> Backend API for CarriersApp application

## Usage

Rename "config/cpnfig.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
